ЧПОКЕР — медиа-ассеты
=====================

Страница работает и без файлов (есть CSS-фолбэки), но если положить сюда
ваши материалы, они автоматически появятся на странице:

  assets/poker.mp4   — фоновое видео в герое (то самое со стола с мониторами)
  assets/dealer.jpg  — фото дилера (ч/б портрет)

Имена файлов должны быть ровно такими. Поддерживается mp4 для видео и jpg для фото.
